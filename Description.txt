This Face Mask Detection (FMD) System was developed using Python, Deep Learning (CNN), OpenCV, and Arduino to promote public health and safety. The system analyzes real-time video streams to detect whether individuals are wearing masks and provides instant feedback using LED indicators and buzzer alerts.
